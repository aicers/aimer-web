# PDF Fonts

Place the following WOFF2 font files in this directory for PDF
generation to work correctly:

- `Roboto-Bold.woff2`
- `Roboto-Medium.woff2`
- `Roboto-Regular.woff2`
- `Pretendard-Bold.woff2`
- `Pretendard-Medium.woff2`
- `Pretendard-Regular.woff2`

Roboto is available from [Google Fonts](https://fonts.google.com/specimen/Roboto).
Pretendard is available from [GitHub](https://github.com/orioncactus/pretendard).
